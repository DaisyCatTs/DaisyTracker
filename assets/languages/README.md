# Language Assets

These PNG files are curated DaisyTracker assets used as optional Discord embed thumbnails.
They are distributed with this repository under the same `GPL-3.0-only` license as the
project unless a future file documents a narrower license next to this README.

DaisyTracker detects many more languages than it has local icons for. Missing icons are
intentional: the embed falls back to the repository/avatar imagery and still uses the
detected language color.

Do not add third-party logo files here unless the source, license, and trademark status
are documented in this file before release.
